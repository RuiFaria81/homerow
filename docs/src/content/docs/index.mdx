## How It Works

The project has five parts working together:

1. **Infrastructure provisioning** (`infra/`) — creates VPS, DNS records, and backup storage using Terraform.
2. **Server configuration** (`flake.nix`, `modules/`) — turns the VPS into a NixOS mail host.
3. **Sync layer** (`sync-engine/`) — keeps mailbox data synchronized into PostgreSQL for fast UI queries.
4. **Web interface** (`webmail/`) — custom frontend/backend for daily email workflows.
5. **Deploy orchestration** (`scripts/`) — connects everything into one repeatable flow.
